# GMX Contracts
Contracts for GMX.

## Install Dependencies
If npx is not installed yet:
`npm install -g npx`

Install packages:
`npm i`

## Compile Contracts
`npx hardhat compile`

## Run Tests
`npx hardhat test`


## USEFUL COMMANDS
```
npx hardhat run scripts/peripherals/read.js --network arbitrum
npx hardhat test test/mirror/mirrorTrading.js
```
